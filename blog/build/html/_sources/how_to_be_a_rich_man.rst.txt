1.2 安装Python
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Python官网：
python.org
点击主页的 Downloads 选择 Windows 右边部分点击一下 Python 3.x.x 即可下载Python的最新版本了！

双击下载好的安装包：
注意:
一定要在安装的时候勾选复选框 "Add Python 3.x to PATH",
来自作者的善意翻译：将Python3.x 加入到系统环境变量"PATH"中
如果我不勾选会怎么样？
你的Python将无法在命令提示符(CMD)中运行，你无法使用非常Nice的PIP来安装库
点击复选框后，直接点击这句：Customize Installation(自定义安装)
然后，会让你配置选项，直接默认，然后点击Next,进入安装自定义安装Python的安装界面
下方的Customize install location这句是安装路径,我个人安装在D盘的Python文件夹下,所以下方的路径我选择的是D:\Python
(D盘下的Python文件夹)
上面的复选框吗！
小孩子才做选择，我全都要！
(将复选框全部勾选)
注意：
最上方的复选框选不选凭心情
如果你选了
安装在你电脑的全部用户里
不选？
只安装在你的当前用户里
然后点击Install就开始安装了！
